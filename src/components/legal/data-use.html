Data Use and Privacy Overview

Last updated: June 9, 2026

Privacy Mode ON

When Privacy Mode is enabled:

  • Customer Data will not be used for training by ZOD
  • ZOD maintains zero data retention (ZDR) agreements with all providers
  • AI model providers will not store or train on your data
  • Model providers may run risk classifiers to detect violations
  • Non-ZDR models designated as such or require admin opt-in

Privacy Mode OFF

When Privacy Mode is disabled:

  • We may use and store codebase data, prompts, editor actions, code snippets, and other code data and actions to improve our AI features and train our models
  • Prompts and limited telemetry may be shared with model providers when you explicitly select their models
  • Inference providers (Baseten, Together AI, Fireworks) may temporarily access and store model inputs/outputs to improve inference performance; data deleted after use

Other important notes

  • Even with your API key, requests go through ZOD backend for final prompt building
  • Codebase indexing: code uploaded in small chunks to compute embeddings; plaintext code ceases to exist after the request; embeddings and metadata (hashes, file names) may be stored
  • File caching: temporary, encrypted with unique client-generated keys, never permanently stored, never used as training data when privacy mode is enabled
  • Data not shared with model providers if account created before Oct 15, 2025
