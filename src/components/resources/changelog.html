4.0 Jun 22, 2026

Cloud agents on mobile

Cloud agents on mobile, Remote Control, Live Activities and push notifications, Artifacts and SCM, Mobile app download.

  • Cloud agents on mobile - Open the Zod mobile app, choose a repo, and launch an agent the same way you would on the desktop app. Pick any frontier model, describe ideas out loud with voice input, and use slash commands.
  • Remote Control - Use Remote Control in the Agents Window to take an agent you are running on your computer and keep directing it from your phone.
  • Live Activities and push notifications - Track the status of your agents with Live Activities on your lock screen. Get push notifications when an agent finishes, needs input, or is ready for review.
  • Artifacts and SCM - Review demos, screenshots, logs, and diffs from your phone. Leave follow-up instructions, or merge the PR directly from the app.
3.9 Jun 22, 2026

Customize Zod

Plugins, skills, MCPs in one place, Marketplace leaderboard, Plugin canvases.

  • Plugins, skills, MCPs in one place - The new Customize page brings plugins, skills, and MCPs into a single view.
  • Marketplace leaderboard - Zod now shows you a leaderboard of the most popular plugins, skills, and MCPs across your team.
  • Plugin canvases - Plugins now support prebuilt canvases. Hex Canvas for data visualizations, Atlassian Canvas for realtime view of issues/projects/documents.
3.8 Jun 18, 2026

Improvements to Zod Automations

  • /automate skill - Create an automation directly in your local agent session.
  • Emoji trigger for Slack - React to any Slack message with a designated emoji to kick off an automation.
  • 5 new GitHub triggers - Issue comment, PR review comment, PR review submitted, Review thread updated, Workflow run completed.
  • Computer use tool for automations - Cloud agents kicked off by automations can now use their own computers to produce demos or artifacts.
3.7 Jun 17, 2026

Cloud Environment Setup and Cloud Subagents

  • Cloud environment setup - Zod can help you set up your dev environment in the cloud in less than 10 minutes.
  • Cloud subagents with /in-cloud - Spin up a cloud subagent in its own VM.
  • Handoff between local and cloud - Move agent sessions more reliably between your local computer and the cloud.
3.6 Jun 10, 2026

Bugbot is now over 3x faster, 22% cheaper, and finds 10% more bugs

Bugbot powered by Composer 2.5, /review before push, Only review what is new.

  • Bugbot powered by Composer 2.5 - Average review time around 90 seconds (down from around 5 minutes).
  • /review before push - Run Bugbot before you push with /review.
  • Only review what is new - Configure Bugbot to only review what is new since the last review.
