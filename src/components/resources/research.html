We're an applied research team focused on building AI that helps developers ship ambitious software faster and more reliably than ever before. Our research spans model training, systems engineering, and human-computer interaction.

Research areas

Codebase indexing

Custom embedding models that index entire codebases with high recall. Agents get instant access to relevant code, documentation, and context across millions of lines.

Semantic search for code

State-of-the-art semantic search that understands code semantics, not just syntax. Ask questions in natural language and find exactly what you need.

Reinforcement learning for agents

Training coding agents with RL on real-world scenarios. Our agents learn from millions of successful coding sessions to make better decisions.

Multi-agent collaboration

Systems that enable multiple AI agents to work together on complex tasks, dividing work, sharing context, and collaborating in parallel.

Tab prediction models

Specialized transformer models trained for sub-100ms code completion. Our Tab model predicts multi-line edits with industry-leading accuracy.

Privacy-preserving AI

Techniques for running AI models without compromising user privacy. Including zero data retention, on-device processing, and differentially private training.
